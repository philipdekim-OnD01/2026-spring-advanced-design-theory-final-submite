## Project Title

RSP_SSD_LAB: Raspberry Pi SSD Object Detection Practice

## Summary

This final work is a Raspberry Pi object detection lab that runs rock-paper-scissors hand-shape detection with an SSD-based TFLite model. It also compares model size, inference time, CPU usage, and RAM usage before and after TFLite quantization.

## Included Topics

- Raspberry Pi real-time camera inference
- SSD object detection with TFLite
- Rock-paper-scissors hand-shape classes
- Float32 and int8 TFLite model comparison
- Benchmarking model size, inference speed, CPU usage, and memory usage
