## 주요 내용
### 파인튜닝 방법 (바위 인식 개선)
- **DenseNet121 완전 동결** — ImageNet 범용 특징 유지로 실환경 일반화
- **Dropout 0.5** — 과적합 억제 (기존 0.2)
- **class_weight 바위 2.0** — 바위 오분류 시 손실 2배 반영
